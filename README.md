# Data Professional Survey Dashboard
## Overview
This project delivers a comprehensive look into the global data science industry by processing 600+ professional survey responses within Power BI. The workflow leverages Power Query for advanced data transformation—specifically splitting complex string delimiters and normalizing inconsistent salary ranges—to build a structured data model. The final dashboard features Gauge Charts for sentiment tracking and Tree Maps for geographic insights, offering a definitive benchmark for salary expectations and tool preferences in the current market.
